📸 Aperçu de l'accueil

📥 Installation et lancement

  1. Téléchargez le fichier CEMSC.zip via le bouton ci-dessus.
  2. Faites un clic droit sur le fichier téléchargé et sélectionnez "Extraire tout..." (ou utilisez 7-Zip / WinRAR).
  3. Ouvrez le dossier extrait, puis ouvrez le sous-dossier dist.
  4. Double-cliquez sur CEMSC.exe pour lancer le programme.
  5. Si Windows Defender affiche un avertissement, cliquez sur "Informations complémentaires" puis "Exécuter quand même" (le logiciel est sûr).

💡 Créer un raccourci sur le Bureau

Pour un accès rapide, vous pouvez épingler le programme sur votre Bureau :

  • Dans le dossier dist, faites un clic droit sur CEMSC.exe
  • Choisissez Envoyer vers → Bureau (créer un raccourci)
  • Un raccourci avec l'icône du programme apparaîtra sur votre Bureau

Vous pouvez également faire un clic droit sur CEMSC.exe et sélectionner "Créer un raccourci", puis déplacer ce raccourci où vous le souhaitez.

📁 Où sont sauvegardées mes données ?

Les profils, scores et statistiques sont automatiquement enregistrés dans le dossier :
%LOCALAPPDATA%\CentreEvaluation
(accessible en copiant ce chemin dans l'explorateur de fichiers). Vous pouvez donc déplacer le programme ou le raccourci sans perdre votre progression.

🎯 Modules d'entraînement

Speed Click

Testez votre rapidité et précision avec des cibles de taille décroissante. Clics gauche/droite, combo et bonus de rapidité.

Keyboard Master

Apprenez à taper rapidement sur tout le clavier : alphabet, accents, ponctuation, raccourcis et mode All Stars.

Profils multiples

🏆
