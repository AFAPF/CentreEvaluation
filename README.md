CENTRE D'ÉVALUATION - Maîtrise Souris & Clavier
===============================================

Installation :
1. Décompressez ce dossier où vous le souhaitez (ex: Bureau, Program Files).
2. Lancez CEMSC.exe (vous pouvez créer un raccourci sur le Bureau).
3. Si Windows SmartScreen affiche un avertissement, cliquez sur "Informations complémentaires" puis "Exécuter quand même".

Utilisation :
- Créez un profil (ou plusieurs) pour suivre vos progrès.
- Choisissez un module : Speed Click, Keyboard Master, Atelier Souris, Dactylo.
- Les statistiques sont sauvegardées automatiquement.

Problèmes ?
Consultez la page du projet : https://afapf.github.io/CentreEvaluation/
